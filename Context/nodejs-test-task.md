# Build a Fashion News Monitoring Agent
Create an AI agent which tracks news and updates related to a country's economics. The agent should:

- Monitor relevant sources
- Filter out noise and identify meaningful updates
- Deliver info in a clear, structured format
- Allow to add or remove topics, sources and competitors that it monitors

You’re free to choose tools, sources, and architecture. We care more about how you think and design the solution than feature complexity.

## Deliverables

1. **Working prototype** - a simple bot/agent showing the core workflow (MVP is enough).
2. **Loom walkthrough** - explain how it works, your key decisions, tools used. Mention any additional services/APIs you’d use in production and key limitations of your prototype.
3. **Further development** - prepare a small note highlighting how you’d improve this tool in the future. What other features you’d add, optimizations you’d set up. Define the timeframe and key improvements you’d want to deliver.
